3-Qadamsettings.py sozlamalari

Djangoga loyihadagi static (bizning fayllar) va media (rasmlar yuklanadigan joy) qayerda ekanligini aytishimiz kerak.

core/settings.py
import os
# Static files (CSS, JavaScript, Images)
STATIC_URL = 'static/'
STATICFILES_DIRS = [os.path.join(BASE_DIR, 'static')]
# Media files (Foydalanuvchilar va Admin yuklaydigan rasmlar)
MEDIA_URL = 'media/'
MEDIA_ROOT = os.path.join(BASE_DIR, 'media')

4-Qadamurls.py da fayllarni ochish

Fayllar joyini ko'rsatdik, lekin Django ruxsatsiz ularni brauzerga bermaydi. URL oxiriga + static(...) qoamoq shart!

from django.conf import settings
from django.conf.urls.static import static

urlpatterns = [
path('admin/', admin.site.urls),
path('', include('store.urls')),
] + static(settings.MEDIA_URL, document_root=settings.MEDIA_ROOT)
# Ushbu qator bazadan kelayotgan (mahsulot) rasmlarni brauzerda ko'rsatishga ruxsat beradi!

Tuzatish6-Qadam: Jinja ulaymiz

Barcha eski manzillarni Django (Jinja) teglariga almashtiramiz.

{% load static %} # HTML faylning eng tepasiga qo'shish shart!
<!DOCTYPE html>
<head>
<!-- XATO (Eskisi): <link rel="stylesheet" href="assets/css/style.css"> -->
<!-- TO'G'RI (Yangisi): -->
<link rel="stylesheet" href="{% static 'css/style.css' %}">
</head>
<body>
<!-- Rasm ulash -->
<img src="{% static 'images/logo.png' %}">

</div>

{% endfor %}
